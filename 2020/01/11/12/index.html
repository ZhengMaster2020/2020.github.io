前端日常开发碰到的CSS基础问题

1. CSS的常见布局

  • flex布局（日常开发中用得最为频繁的布局）
  • grid布局（网格布局，主流浏览器兼容性差）
  • 定位布局（基于position的relative与absolute进行布局）
  • 浮动流布局 （基于float属性）
  • 文档流布局（基于div元素）
  • 圣杯布局
  • 双飞燕布局
  • 水平垂直居中布局
2. CSS的盒子模型
  • 标准盒子模型
  • IE盒子模型
3. CSS的选择器以及权重
  • css解析规则（css选择器是从右到左，从上到下进行解析）
  • 优先级计算规则
    • 最近的祖先样式比其他祖先样式优先级高
    • “直接样式”比”祖先样式”优先级高
    • 内联样式 > ID 选择器 （权重100）> 类选择器 （权重10）= 属性选择器 = 伪类选择器 > 标签选择器 （权重1）= 伪元素选择器
    • !important 拥有最高优先级，!important 比内联优先级高
4. CSS3新特性 Link

  • 过渡

    (transition： CSS属性，花费时间，效果曲线(默认ease)，延迟时间(默认0)

  • 动画

    animation：动画名称，一个周期花费时间，运动曲线（默认ease），动画延迟（默认0），播放次数（默认1），是否反向播放动画（默认normal），是否暂停动画（默认running））

  • 形状变换

    transform: rotate(30deg)

  • 新增选择器

  • 阴影

    box-shadow: 水平阴影的位置 垂直阴影的位置 模糊距离 阴影的大小 阴影的颜色 阴影开始方向（默认是从里往外，设置inset就是从外往里）;

    box-shadow: 10px 10px 5px #888888;

  • 边框

    边框图片：border-image

    边框圆角：border-radius

  • 背景

    指定背景描绘区域： background-clip（border-box，padding， content-box）

    指定background-position属性为相对位置：background-origin（border-box，padding， content-box）

    指定背景的大小：background-size（contain，cover，auto， center）

  • 反射

    倒影

    -webkit-box-reflect:方向[ above-上 | below-下 | right-右 | left-左 ]，偏移量，遮罩图片

  • 文字

    文字阴影

    text-shadow:水平阴影，垂直阴影，模糊的距离，以及阴影的颜色

    换行

    word-break: normal|break-all（单词内换行）|keep-all

    word-wrap: normal|break-word

    省略号

    text-overflow：clip|ellipsis|string

  • 颜色

    rgba（rgb为颜色值，a为透明度）

    h:色相”，“s：饱和度”，“l：亮度”，“a：透明度”

    • 渐变(Gradient)Link
  • 滤镜(Filter)

  • 媒体查询 (@media screen)

  • 弹性布局（flex）Link
  • 栅格布局（grid）Link

​ 不可继承的样式：border, padding, margin, width, height
