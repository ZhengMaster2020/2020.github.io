变量、局部变量与全局变量

  • JavaScript 变量是存储数据值的容器， 在以前的版本通常是使用var来声明一个变量

  • js中的变量可以分为局部变量(local variable )和全局变量(global variable ),全局变量是所有对象函数数组等数据类型无论在什么时候都可以访问得到这个全局变量，而局部变量一般只有在一定的范围内才可以访问得到，js中通常是在一个函数内定义一个变量就是局部变量，这个变量可以在这个函数内的其他位置都可以访问得到，但是函数外部却不能直接访问得到函数内部变量，也就是函数内部的变量对函数外来说的不可见的。

  • 一般没有使用var关键字来声明的变量都是作为全局变量，在客户端的js中全局变量都可以通过window对象来访问得到这变量，而这个全局变量其实也是window对象的属性而存在的。

变量作用域以及作用域链

  • ECMAScript2015版本以前都是没有块级作用域的说法，但是ECMAScript2015(民间俗称es6)出来以后引入了let与const这两个关键字可以实现块级作用域的效果。所谓的块级作用域就是一个{}来作为作用域的分界线，大括号外面的其他程序访问不了大括号里面变量等

  • 任何变量（函数也视为变量），如果没有在当前函数作用域中找到，就会继续往上查找，最后如果在全局作用域中也没有找到，则报ReferenceError错误

  • 如果两个不同的函数各自申明了同一个变量，那么该变量只在各自的函数体内起作用。换句话说，不同函数内部的同名变量互相独立，互不影响

  • js的变量作用域是基于作用域链的，就是如果其他程序需要用到这个变量，但是却在函数、对象等内部的变量却找不到这个变量，它就会一层一层往外找，查找其他地方是否有这么一个变量，如果有就使用这个变量，倘若一直查找到最顶层都没有这个变量就返回Undefined

  • 变量的作用域可以是全局作用域、函数作用域和块级作用域，js变量会存在变量声明提升这个问题，以及在for进行循环时它是不能拥有块级作用域的，所以可以使用es6的let来声明循环变量

js的数据类型以及它们的类型检查

  • js的数据类型(7种)可以分为基本数据类型和引用数据类型，其中基本数据类型可以分为以下5大类

Number、Boolean、String、Null、Undefined、Symbol，而引用类型是Object（而Object又包括有Array、Function、RegExp、Date

  • Symbol 是 ES6 引入了一种新的原始数据类型，表示独一无二的值

  • 基本数据类型的变量值存储在内存栈中，引用类型的变量值存储在内存堆中

  • undefined 这个值表示变量不含有值。可以通过将变量的值设置为 null 来清空变量，null表示一个“空”的值。例如Java也用null，Swift用nil，Python用None表示。但是，在JavaScript中，还有一个和null类似的undefined，它表示“未定义”。JavaScript的设计者希望用null表示一个空的值，而undefined表示值未定义。null的值之前有过，但现在没有了为空；而undefined一开始就不存在，都没有诞生过，值从何而来。

检验

    1. 使用*typeof *操作符来检验基本数据类和Object，返回一个表示数据类型的字符串，返回结果包括：number、string、boolean、object、undefined、function。
    1. typeof可以对基本类型number、string 、boolean、undefined做出准确的判断（null除外，typeof null===“object”，这是由于历史的原因，而对于引用类型，除了function之外返回的都是object。但当我们需要知道某个对象的具体类型时，typeof就显得有些力不从心了）
    1. instanceof 当我们需要知道某个对象的具体类型时,可以用运算符 instanceof，instanceof操作符判断左操作数对象的原型链上是否有右边这个构造函数的prototype属性，也就是说指定对象是否是某个构造函数的实例，最后返回布尔值
