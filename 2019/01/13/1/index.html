##ES6新特性

1.新增了let、const

A. var定义变量的不足之处：没有块级作用域、定义的常量可以做第二次或者多次改动、存在变量提升  
B. let用来声明一个变量,const用来声明一个常量。
C. let的特性：块级作用域、存在TDZ(temporal dead zone)、不存在变量提升、不允许重复声明
D. const的特性：拥有let特性，只能保证指针固定（即总是指向另一个固定的地址）  
E. const本质：变量指向的那个内存地址所保存的数据不得改动 ,简单数据类型值就保存在变量指向  
的地址内存里（常量）,复合数据类型保存的只是一个指向实际数据的指针

2.解构赋值

A. 数组的解构赋值

let [a,b,c] = [1,2,3];

B. 对象的解构赋值

let {f,s,h} = {f:1, s:2, h:3}

C. 字符串的解构赋值

const [a, b, c, d, e] = ‘hello’

3.函数的扩展

A. 箭头函数(函数的简写)

let sum = (num1,num2) => num1 + num2;
/*一个参数”()”可以省
* 如果只有一个return，”{}”可以省略
**/
let add = num => num += num;

B. 参数的默认值、展开、收集

let arr1 = [1，2，3]
let arr2 = [4, 5, 6]
let arr3 = […arr1,…arr2]
let sum = (one=50, two=55, …args) =>{
console.log( …args);
return one + two;
};
console.log(sum(100,105,88,20,30));

4.字符串的扩展

A. 新增的方法
   startWith(string):字符串以什么字符开头
   endWith(string):字符串以什么字符结束

B. 模板字符串
   使用一对反引号:``     在反引号使用变量：${something} 可以折行

//字符串与模板字符串
let str = ‘somethings just like this’;
console.log(str.startsWith= ${str.startsWith("some", )});
console.log(str,endsWith= ${str.endsWith('this')});

结果：  str.startsWith= true;
        str,endsWith=
        true

5.数组的扩展

A. 新增的方法
   map()方法：映射 返回一个新数组
   reduce()方法：汇总 一堆数据进去出来一个数据结果
   filter()方法：过滤器 符合条件的数据留下来并且返回
   forEach()方法：遍历器 循环遍历输入的数据
B. 扩展运算符（...） 
 主要用于函数调用 一定程度上取代apply方法将数组转换为函数的参数
 将一个数组转为用逗号分隔的参数序列(将数组转换为函数参数)

function add(x, y) {
return x+y=${ x }+${ y }=${x+y}; }
let number = [5,66];
console.log(add(…number));

//es5 的写法
function f(x, y, z) {
       return x+y+z;
    }
    let args = [1, 2, 8];
    let n = f.apply(null, args);
    console.log(n);

 //es6 的写法
 let n =  f(...args);

6.面向对象

A. 写法的不同

ecmascript 6
//es5 的面向对象写法
function User (name,pass){
this.name = name;
this.pass = pass;
}
User.prototype.showMsg = function () {
console.log(this); // 指向的是User的实例化对象 user
console.log(“name=”+ this.name);
console.log(pass=${this.pass}) };
//es5 写法
User.prototype.eat = function (food){
console.log(${this.name} is eatting ${food});
};
//es6 写法
User.prototype.eat = (food,name) => {
console.log(this); //this 指向的是window全局对象
console.log(${name} is eatting ${food});
};
let user = new User(“zhangsan”,123456);
user.showMsg();
user.eat(“hotdogs”, user.name);
// console.log(user.proto === User.prototype); true

B. es6 有了class、extends关键字、构造器，继承的实现变得更简单

User.call(this, name, this.pass);
this.stuNum = stuNum;

}
Student.prototype = new User();

//es6 继承新写法
class Users{
constructor(name, password){
this.name = name;
this.password = password;
}
showMsg(){
console.log(this.name=${this.name});
console.log(this.password=${this.password});
}
}
let user2 = new Users(“xinxin”,1234567);
user2.showMsg();
class Students extends Users{
constructor(name,password,stuNum){
super(name,password);
this.stuNum = stuNum;
}
showNum(){
console.log(this.stuNum = ${this.stuNum});
}
}
let st = new Students(“linling”,55544,161504210);
st.showMsg();
st.showNum();
